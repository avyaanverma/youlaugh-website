Setup Guide

Up and running
in 3 steps.

No store listing. Load it via Developer Mode — it takes about 2 minutes.

01

Download & Unzip

  • Click the Download Extension button above to get the ZIP file from GitHub.
  • Extract it to a folder on your computer — for example: Desktop/YouLaugh
  • Open the extracted folder and confirm it contains manifest.json directly inside (not nested in another folder).
02

Load in Browser via Developer Mode

  • Open your browser's extension page:
    Chrome / Brave / Edge → chrome://extensions
    Firefox → about:debugging#/runtime/this-firefox (temporary load)
  • Toggle on Developer mode (top-right switch in Chrome/Brave/Edge).
  • Click Load unpacked and select the extracted YouLaugh folder.
  • The YouLaugh icon should now appear in your browser toolbar. You may need to pin it from the extensions puzzle-piece menu.
03

Start Downloading

  • Navigate to any YouTube video page.
  • Click the YouLaugh extension icon in your toolbar to open the popup.
  • Select your preferred quality from the dropdown.
  • Click Download and wait — the server will process and deliver the file.
Troubleshooting

Something
not working?

Issue 01
Popup data won't load

Refresh the YouTube tab first, then reopen the extension popup. The extension needs an active YouTube video page to read from.

Issue 02
Download fails on a video

Some videos are restricted at the source platform level. Try a different video. If it consistently fails, it's likely a platform restriction, not a bug.

Issue 03
Extension changes not reflecting

Remove the extension from your browser, then re-load the unpacked folder via Developer mode. Chrome caches extension state aggressively.

Issue 04
No visible download progress

This is expected in v1. Downloads process server-side and may not trigger the browser's native progress UI. Check your Downloads folder after a moment.
